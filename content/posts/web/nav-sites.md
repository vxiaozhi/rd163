整理一份按技术栈归类的开源导航站项目清单，方便挑选合适的方案自建导航页或浏览器主页。

## React / Next.js

- [next-web-nav 一个简单好用的导航网站](https://github.com/liwenka1/next-web-nav):基于 Next.js 14 + Tailwind CSS + shadcn/ui，本地数据持久化，支持可视化数据管理面板。
- [tap4-ai-webui](https://github.com/6677-ai/tap4-ai-webui):面向 AI 工具站的导航模板，内置支付能力。
  > 注：截至核实日，该仓库返回 404，可能已迁移或下架，请以最新链接为准。

## Nuxt

- [geek-navigation 极客猿导航](https://github.com/geekape/geek-navigation):独立开发者导航站，前端 + Server + Admin 三端分离的完整方案。

## Vue

- [naviii 一个简单优雅的导航页](https://github.com/eehhh/naviii):个人设置存储于本地，轻量极简。
- [Dashy](https://github.com/lissy93/dashy):GitHub 约 26K Stars，支持可视化编辑、状态监控、多主题与多语言。

## Angular

- [发现导航](https://github.com/xjh22222228/nav):原支持网站存活状态检测、多款高颜值主题切换、三叉树分类、SEO 友好。
  > 注：截至核实日，该仓库已清空（0 Stars）,"发现导航"可能已迁移或停止维护，请参考作者主页或社区分支。

## Golang + React

- [Van Nav](https://github.com/Mereithhh/van-nav):由 Golang 实现，全平台支持，单文件部署，自带 Admin 管理、搜索以及 Chrome 扩展。

## Golang + Vue

- [迷你网址导航 NavSite](https://github.com/cifaz/nav-site):后端使用 Golang + Gin。
- [Navigation-Web](https://github.com/98XiaoTang/Navigation-Web):基于 Golang 与 Vue 的个人导航网站(原 `tanghancheng/Navigation-Web` 已重定向至此)。

## Golang + Template

- [lczero-server](https://github.com/LeelaChessZero/lczero-server):Leela Chess Zero 的服务端，含站点分发与训练数据收集。
- [go-blog](https://github.com/izghua/go-blog):Golang + Gin + Vue + MySQL 实现的博客网站。
- [eiblog](https://github.com/eiblog/eiblog):基于 Gin 的高性能博客系统。

## Python + Django + Vue

- [WebsiteGuide](https://github.com/mizhexiaoxiao/WebsiteGuide):企业内部网址导航系统，支持网址增删改查、icon 图标替换、用户管理等功能。

## Java + Vue

- [team-nav 网站导航服务](https://github.com/tuituidan/team-nav):前后端完整，后端功能完善，主打轻量级、无额外依赖服务。

## Jekyll

- [BYR-Navi](https://github.com/BYR-Navi/BYR-Navi):北邮人的轻量级、可配置导航站框架。

## ASP

- [小呆导航](https://github.com/xiaodai945/WEBJIKE):支持动态更换主题与登录。

## PHP

- [OneNav](https://github.com/helloxz/onenav):基于 PHP + SQLite 3 的书签管理系统，支持后台管理、Chrome/Firefox/Edge 书签批量导入及多主题。
- [六零导航页 LyLme Spage](https://github.com/LyLme/lylme_spage):支持后台添加链接、自定义搜索引擎，简洁无广告。

## HTML / 静态

- [Tilde](https://github.com/xvvvyz/tilde):面向资深网民的极简浏览器主页，大幅提升搜索效率。
- [Pintree](https://github.com/Pintree-io/pintree):通过 Chrome 扩展一键导出收藏夹生成导航站。
- [Webstack 网址导航](https://github.com/WebStackPage/WebStackPage.github.io):中英文版本 + 丰富生态（Hugo 主题、多语言后端）。
- [Webstack 网址导航 Hugo 版](https://github.com/shenweiyan/WebStack-Hugo):支持 YAML 一键配置的纯静态导航。
- [网址导航（静态页面通用模板）](https://github.com/geneasy/nav):基于 Handlebars 模板语言，页面效果与 Webstack 类似。
- [极速页导航 - 无服务版本](https://github.com/iuv/jisuye):支持在线编辑。
- [Homer](https://github.com/bastienwirtz/homer):GitHub 约 11.5K Stars，基于 Vue 的纯静态主页，通过 YAML 配置，内置搜索。
- [static-nav](https://github.com/haixuxu/static-nav):静态导航主页，集成百度、Google 等搜索引擎。

## 其他

- [蜗牛个人导航](https://github.com/eallion/favorite):Serverless 版个人导航，适配 EdgeOne Pages、Cloudflare Pages、Vercel。
- [一个导航](https://github.com/HoYuenhang/aNavigation):基于 Bootstrap。
- [MyNav 我的导航](https://github.com/j67678/MyNav):仿 Windows 10 桌面风格。
